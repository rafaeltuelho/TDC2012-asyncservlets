Projeto contendo o código da Demo mostrada na palestra do TDC 2012 edicao Goiania-GO.

Trata-se de uma webapp criada para demostrar o comportamento do web conatainer 
em dois cenários:

1) processamento da requicição http utilizando o modo síncrono
2) processamento da requicição http utilizando o modo assíncrono (disponível na API Servlet 3.0)

A aplicação simula um cliente REST do serviço de bookmarks na nuvem Delicious.

Ambiente utilizado:
 - JavaSE 6
 - Netbeans 7.2
 - Glassfish 3.1.2

obs:
  Algumas classes foram geradas pelos wizards disponíveis no Netbeans. Portanto não repare o design da app.
  O objetivo foi demonstrar o suporte ao processamento assícrono oferecido pela API Servlets 3.0.
